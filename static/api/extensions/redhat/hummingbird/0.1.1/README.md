# Hummingbird Extension for Podman Desktop

This extension adds a catalog of Hummingbird alternative images to Podman Desktop.

### Features

- **Catalog of alternatives**: Browse a curated list of Hummingbird images.
- **Local image analysis**: Identify local images that have Hummingbird alternatives.
- **Container hardening**: Clone an existing container and replace its base image with a Hummingbird alternative.
